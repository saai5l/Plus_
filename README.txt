   <link rel="manifest" href="/manifest.json">
   <meta name="mobile-web-app-capable" content="yes">
   <meta name="apple-mobile-web-app-capable" content="yes">
   <meta name="apple-mobile-web-app-title" content="Plus Dev">
   <link rel="apple-touch-icon" href="/Plus_Dev_No_Wellpeper.png">
   <link rel="stylesheet" href="/pwa-styles.css">

✅ في </body> أضيفنا:
   <script src="/pwa-helper.js"></script>
   <script src="ai-chat-enhanced.js"></script>
   <!-- Service Worker Registration -->
   <script>
     if ('serviceWorker' in navigator) {
       navigator.serviceWorker.register('/sw.js', {
         scope: '/'
       }).then(reg => {
         console.log('✅ Service Worker registered:', reg.scope);
       }).catch(err => {
         console.warn('⚠️ Service Worker failed:', err);
       });
     }
   </script>

❌ المشكلة: لا يظهر زر "ثبّت الآن"

✅ الحلول:
   1. تأكد أن الموقع على HTTPS (ضروري جداً!)
   2. افتح DevTools (F12)
   3. اذهب إلى Application > Service Workers
   4. إذا كان أحمر = اضغط Unregister ثم أعد تحميل الصفحة
   5. انتظر 5 ثواني وحاول مرة أخرى

──────────────────────────────────────────────────────────────────────────

❌ المشكلة: Service Worker ظهر أحمر

✅ الحل:
   1. اضغط Unregister
   2. أعد تحميل الصفحة (F5)
   3. افتح DevTools مرة أخرى
   4. يجب يصير أخضر "activated"

──────────────────────────────────────────────────────────────────────────

❌ المشكلة: Offline لا يعمل

✅ الحل:
   1. F12 > Network > اختر "Offline"
   2. أعد تحميل (F5)
   3. يجب تعمل الصفحة
   4. إذا لا تعمل = شغّل Hard Refresh (Ctrl+Shift+R)
